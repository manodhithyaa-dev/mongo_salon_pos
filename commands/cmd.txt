Git Commands:

1. push code
git add .
git commit -m "<statement>"
git push

2. pull code
git pull origin main

3. check status
git status

4. set origin
git remote add origin <repository-url>

5. set branch as main
git branch -M main

6. verify origin
git remote -v

7. first push (only once)
git add .
git commit -m "<statement>"
git push -u origin main


Docker Commands:

1. Create environment
docker compose up --watch

2. Create environment (background)
docker compose up -d

3. Delete environment
docker compose down

4. Check active containers
docker ps

5. Check all containers
docker ps -a

6. View logs
docker compose logs -f

7. Rebuild containers
docker compose up --build

8. Restart containers
docker compose restart


PM2:

1. start service
pm2 start "docker compose up --watch" --name salon-pos

2. stop service
pm2 stop salon-pos
or
pm2 stop <index number>

3. restart service
pm2 restart salon-pos
or
pm2 restart <index number>

4. check logs
pm2 logs salon-pos
or
pm2 logs <index number>

5. see index number and list all processes
pm2 list

6. save process list
pm2 save

7. remove process
pm2 delete salon-pos


Install Packages:

1. Install new packages
npm install <package-name>
or
npm i <package-name>

2. Initial setup after clone
npm install
(inside both frontend and api folders)

3. Install package and save to package.json
npm install <package-name>

4. Install package globally
npm install -g <package-name>
