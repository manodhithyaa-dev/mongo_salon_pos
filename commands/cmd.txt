Git Commands:

1. push code
git add .
git -m commit "<statement>"
git push -u origin main

2. pull code
git pull origin main

Docker Commands:

1. Create environment
docker compose up --watch

2. Delete environment
docker compose down

3. Check active containers
docker ps

4. Check all containers
docker ps -a

PM2:

1. start service
pm2 start "docker compose up --watch" --name salon-pos

2. stop service
pm2 stop salon-pos
or
pm2 stop <index number>

3. check logs
pm2 log salon-pos
or
pm2 log <index number>

4. to see index number and list all process
pm2 list

Install Packages:

1. Install new packages
npm install <package name>
or
npm i <package name>

2. Initial Setup after Clone
npm install (inside both frontend and api folder)
